# Assets Directory

This directory should contain the following files for your portfolio:

## Required Images:
- `profile.jpg` - Your profile photo (recommended: 400x400px, professional headshot)
- `project1.jpg` - Image for SAHARA project (recommended: 600x400px)
- `project2.jpg` - Image for Data Structures project (recommended: 600x400px)
- `project3.jpg` - Image for Robot project (recommended: 600x400px)

## Optional Files:
- `resume.pdf` - Your resume/CV file for download
- `favicon.ico` - Website favicon
- Additional project images

## Image Guidelines:
- Use high-quality images
- Optimize for web (compress to reduce file size)
- Maintain consistent aspect ratios
- Use professional, clear photos

## How to Add Images:
1. Replace the placeholder image paths in `index.html`
2. Add your actual images to this directory
3. Update the `src` attributes in the HTML to match your image filenames

## Placeholder Images:
You can use placeholder services like:
- https://via.placeholder.com/400x400 (for profile)
- https://via.placeholder.com/600x400 (for projects)

Or use free images from:
- Unsplash.com
- Pexels.com
- Pixabay.com
