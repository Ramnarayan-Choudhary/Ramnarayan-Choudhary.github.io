LLM Research

Inference-time Steering in LLMs

Investigated inference-time activation steering as a unifying framework for controlling reasoning and alignment behaviors in LLMs. Designed contrastive steering methods that intervene in transformer hidden representations to modulate behavior without parameter updates.

LLMs Activation Steering Alignment
Privacy Research

PromptGCG: Gradient-based Prompt Sanitization

Proposed a gradient-based prompt sanitization method that optimizes private token replacements to minimize privacy leakage while preserving task utility. Developed a formal privacy–utility evaluation framework with adversarial inference attacks.

Privacy Prompt Engineering Adversarial

Machine-Generated Code Detection

Investigated machine-generated code detection under cross-language and multi-generator settings. Analyzed how different adaptation regimes (full fine-tuning, PEFT, prompting) affect semantic vs. superficial cue capture in code provenance classification.
