## ğŸš€ Getting Started

### Prerequisites
- A modern web browser
- Git (for version control)
- Basic understanding of HTML/CSS/JavaScript (for customization)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ramnarayanchoudhary/portfolio.git
   cd portfolio
   ```

2. **Add your assets:**
   - Add your profile photo as `assets/profile.jpg`
   - Add project images to the `assets/` folder
   - Add your resume as `assets/resume.pdf`

3. **Customize content:**
   - Update personal information in `index.html`
   - Modify colors and styling in `styles.css`
   - Add your project links and details

4. **Open in browser:**
   ```bash
   # Simply open index.html in your browser
   open index.html
   ```
